# 项目介绍

manager-api 该项目基于SpringBoot框架开发。

开发使用代码编辑器，导入项目时，选择`manager-api`文件夹作为项目目录

# 开发环境
JDK 21
Maven 3.8+
MySQL 8.0+
Redis 5.0+
Vue 3.x

# 接口文档
启动后打开：http://localhost:8002/xiaozhi/doc.html

# 项目初次启动说明
1. 创建数据库 xiaozhi_esp32_server（创建之后，暂不需要创建数据库表）
2. 修改数据库配置参数
3. 直接启动项目（启动时，会自动创建数据库表）
