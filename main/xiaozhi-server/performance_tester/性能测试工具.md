

# 模型测试工具

======================================================================================
## 1.在main/xiaozhi-server/performance_tester目录下创建data目录

## 2.在data目录下创建.config.yaml文件 
performance_tester目录下创建data目录中的.config.yaml文件会在测试文件运行时优先加载

## 3.在.data/config.yaml中，写入你的语音识别、大语言模型、流式语音合成、视觉模型的参数
### model_dir 是模型所在的文件夹路径，这个路径是相对于所运行的py文件而言，是个相对路径。
### api_key 和 access_token 是必须要有的参数，本地模型的话，这2个参数可以设置为XXXX

```
ASR:
  FunASR:
    type: fun_local
    # 从performance_tester_asr.py文件出发找models的路径
    model_dir: ../models/SenseVoiceSmall
    output_dir: tmp/
    api_key: XXXXX
    access_token: XXXXX
```
